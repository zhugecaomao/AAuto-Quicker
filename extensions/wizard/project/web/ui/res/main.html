所有HTML页面请放置于res目录下.
如果res目录设为内嵌资源,则发布时无需修改任何代码自动生成独立EXE文件。
并在head节点链接"res/js/aauto.js"(更改为页面相对路径),例如在本页添加如下代码：
<script type="text/javascript" src="js/aauto.js"></script>

aauto.js的作用是在所有html文件载入时,根据路径自动载入相应的用户库
例如 "/res/main/html" 加载时 自动调用external.loadServer()接口函数,加载 server.main.aau,
在页面退出时,自动调用 server.main().onUnload() 函数(如果定义了该函数).

aauto.js 同时定义了 console.log() 函数,用于在javascript输出调试信息到控制台

在任意javascript中都可以使用 external.config.配置文件名.配置字段名 直接读写
配置文件, 配置文件将会自动创建并自动保存

推荐的文件存储位置:
所有图片放在/res/image目录下,图片相关的CSS文件放在同级目录下.
所有js文件请置于res/js目录下,jQuery请置于以下位置: "/res/js/jQuery.min.js"，

使用requireJS支持AMD模块加载:
requireJS请置于以下位置: "/res/js/require.js"，
requireJS模块加载入口文件: "/res/js/main.js"，
